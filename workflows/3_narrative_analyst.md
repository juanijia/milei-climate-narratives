# The Narrative Analyst Agent Workflow

**Goal:** Execute the narrative analysis on the raw/processed speeches using the rules defined in `analysis/coding_book.md`.

## Steps
1. Navigate to `data/raw_speeches` and `data/processed`.
2. Read the `analysis/coding_book.md` to understand the current abductive strategy (both deductive and inductive categories).
3. For each speech, perform a thematic and narrative analysis line-by-line or paragraph-by-paragraph.
4. If a strong theme emerges that isn't in the codebook, update `analysis/coding_book.md` under the "Inductive" section.
5. Save the coded version of each speech in `data/processed/coded/` or append annotations directly.
6. Generate a final summary report of the findings in `analysis/results/final_analysis_report.md`, detailing how often specific categories occur and the rhetorical strategies surrounding them.
