# The Methodologist Agent Workflow

**Goal:** Understand the proposed methodology and update the `coding_book.md` with refined or newly extracted categories.

## Steps
1. Navigate to the `docs/methodology` and `docs/references` folders.
2. Use PDF parsing or semantic search tools to read the methodology files.
3. Review the `docs/abstract` file to understand the overall research goal.
4. Read the existing `analysis/coding_book.md`.
5. Propose modifications or additions to the deductive categories in the codebook based on the academic references provided, aligning with an abductive approach.
6. Write a summary of your findings to `analysis/results/methodology_summary.md`.
